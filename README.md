# Secure Command Communication System (RSA + HMAC)

A secure client-server communication system implemented in Python using RSA for secure key exchange and HMAC-SHA256 for message authentication and integrity verification.

---

## Features

- RSA-based secure session key exchange
- HMAC-SHA256 message authentication
- Counter-based replay attack prevention
- Adaptive key evolution
- Replay attack simulation using Scapy
- Concurrent server architecture

---

## Technologies Used

- Python
- Cryptography Library
- Socket Programming
- HMAC / SHA-256
- Scapy
